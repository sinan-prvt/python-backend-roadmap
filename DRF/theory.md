# DRF Theory Notes


## What is DRF

It stands for Django Rest Framework.

It is a powerful and flexible toolkit for building APIs in Django.

It allows Django apps to serve data in JSON, XML, or other formats. 

It supports serialization, validation, authentication, permissions, filtering, pagination, and viewsets.
